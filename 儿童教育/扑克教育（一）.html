# 扑克教育（一）

利用扑克教会小朋友区分颜色、形状，认识数字大小，并学会排序、分类和简单判断。

# 认识颜色

认知扑克牌有红色和黑色两种颜色。 练习：将牌打乱，分为红色黑色两类。

# 认识花色

认识扑克有红桃、方块、梅花、黑桃四种花色。 练习：将牌打乱，分为红桃、方块、梅花、黑桃两类。

# 认识数字

认识数字1~10。 练习：学会将扑克按照1~10顺序/逆序排序。

# 大小规则

知道非王牌中3最小，2最大，可以从小到大排序。 练习：学会将扑克按照3~王牌顺序/逆序排序。

# 综合练习

# 深度比较

  1. 随意抽2张以上的牌，比较大小。
  2. 相同大小的牌，红色更大。
  3. 相同大小颜色的牌，按照黑桃>红心>方块>梅花顺序判断大小。

# 综合排序

  1. 将牌打乱，按照黑桃、红心、方块、梅花顺序分类，并从小到大排列。

# 逻辑判断

  1. 随机抽出1张牌，判断牌堆里还剩几张牌？分别是什么花色？
  2. 牌分为两堆，根据手里的牌，判断对方的牌有什么？

# 记忆训练

  1. 每抽一张牌让孩子记忆一下，连抽3张后，要求按顺序复述是什么牌。
